# Nip Strategy (Nip)

オセロのルールをベースに、盤面を「円形」に進化させた戦略ボードゲームです。
円形だからこそ生まれる、従来のオセロにはない予測不能な展開を楽しめます。

## 🎮 ゲームの概要：オセロとの違い
基本的な「相手の石を挟んでひっくり返す」ルールはオセロと同じですが、**盤面が円形（環状）**である点が最大の特徴です。

### 1. 「角」が存在しない
円形のため、オセロで最も重要な「四隅の角を取る」という概念がありません。どこまで進んでも終わりがない、独特の緊張感が味わえます。

### 2. ループするライン
石を置いたとき、盤面を一周回って裏返せるラインが発生します。「端っこだから安心」という常識が通用しない、360度全方位への注意が必要です。

### 3. CPU対戦
初心者から上級者まで楽しめるCPU思考ルーチンを搭載。円形盤面特有のセオリーを見つけ出し、勝利を掴み取ってください。

## 📲 インストール手順
1. 上記の[ダウンロードリンク](https://github.com/chikuboya/Nip/releases/tag/v1.0.0)を開きます。
2. Assets内にある `nip-0.1-install.apk`を保存します。
3. Android端末でファイルを開き、インストールを実行してください。
   ※「不明なアプリ」の警告が出た場合は、一時的に許可設定を行ってください。
