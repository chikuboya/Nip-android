# Nip Strategy (Nip)

Nip Strategy は、シンプルながら奥が深い戦略ボードゲームを楽しめるAndroidアプリです。
CPUとの対戦を通じて、戦略を練る楽しさを体験してください。

## 🎮 ゲームの概要
このアプリは、盤面上のコマを動かして相手を追い詰める戦略ゲームです。

### 主な機能
- **CPU対戦:** 初級から上級まで、自分のレベルに合わせて挑戦できます。
- **直感的な操作:** タップだけで簡単にコマを動かせます。

## 📲 インストール方法
1. 上記のリンクから `app-release.apk`（またはそれに類するファイル名）をダウンロードします。
2. Android端末でファイルを開きます。
3. 「不明なアプリのインストール」の警告が出た場合は、一時的に許可をしてインストールを続行してください。
